Brian Kernighan

Brian Kernighan contributed to the development of the Unix operating system in the early 1970s. He is a co-creator of the AWK programming language and an author of several books.

Some of my favorite books he's authored include:

  • The Elements of Programming Style (1974, 1978) with P. J. Plauger
  • The Unix Programming Environment (1984) with Rob Pike
  • The Practice of Programming (1999) with Rob Pike
  • The Go Programming Language(2015) with Alan Donovan[14]
  • UNIX: A History and a Memoir(2019)
